# Improtango Website - Claude Context

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Vite
- **Backend**: Convex (forms, rate limiting)
- **Router**: React Router v6
- **Deployment**: Vercel

## Key Features

- Bilingual (Finnish/English) with complete i18n
- Secure forms with rate limiting (1/hour per email)
- XSS protection and input validation
- Teal gradient design theme
- Glass morphism effects

## Commands

- `bun run dev` - Development server
- `bun run build` - Production build
- `bunx convex dev` - Backend development

## Status

Production-ready with security, translations, and modern design complete.
